# 📋 TaskFlow Dashboard

## Objective

The objective of this project is to create a responsive To-Do Dashboard that helps users organize and manage their daily tasks efficiently. The application allows users to add, edit, delete, and complete tasks while tracking their progress through a simple and interactive interface.



## Steps Performed

1. Designed the dashboard layout using HTML.
2. Styled the application with CSS to create a modern and responsive user interface.
3. Implemented task management functionalities using JavaScript.
4. Added options to create, edit, complete, and delete tasks.
5. Integrated priority levels (High, Medium, and Low) for better task organization.
6. Used LocalStorage to save task data and retain it after page refreshes.
7. Developed task statistics to display total, pending, and completed tasks.
8. Tested the application for responsiveness and functionality across different devices.



## Tools Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser LocalStorage API
* Visual Studio Code
* GitHub



## Outcome

Successfully developed a fully functional TaskFlow Dashboard that enables users to manage tasks effectively. The application provides an intuitive user experience, real-time task tracking, priority management, and persistent data storage through LocalStorage, making it a practical productivity tool.



## Features

* Add Tasks
* Edit Tasks
* Delete Tasks
* Mark Tasks as Completed
* Priority-Based Task Management
* Task Statistics Dashboard
* LocalStorage Data Persistence
* Responsive Design
